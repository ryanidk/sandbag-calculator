Sandbag Wall Calculator

Calculate the number of sandbags needed to build a sand wall for flood prevention.

Notes:

  • Only fill each bag two-thirds of the way.
  • The base layer should be a minimum of 4 bags wide, overlapping the sides slightly.
  • Make a pyramid, so each layer has 1 less row.
  • A 10m/33ft long wall, 0.6m/2ft high, would require approximately 360 bags.
  • This tool assumes bags are 36cm/14" x 66cm/26".
